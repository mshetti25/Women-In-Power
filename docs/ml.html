Summary

The 2019 data was examined using both linear regression and two different types of machine learning.

The data was analyzed using linear regression to determine whether the labor force could affect women in managerial positions.

We used Scikit-Learn to conduct a linear machine learning on the dataset.

Logistic machine learning was used to examine how women in parliament and women in the labor force could predict the income grouping of a country.
