Manager vs. Labor Linear Regression

Manager vs. Labor Linear Regression

A linear regression was conducted on the percentage of women in the labor force (x) and the percentage of women managers (y). To avoid examining null values, this was only analyzed in countries that had a percentage of women managers greater than 0. From the scatterplot, we can see that there is a positive correlation between women in the labor force and women managers. In order to more closely examine this, machine learning was conducted.
